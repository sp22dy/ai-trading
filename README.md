# 한국투자증권 Open API 삼성전자 자동매매 프로그램 (모의투자 전용)

한국투자증권 Open API(REST)를 활용하여 삼성전자(005930) 주식을 자동으로 매매하는 파이썬 프로그램입니다.

> ⚠️ **현재 상태 및 이슈**
> *   현재 프로그램의 기본 API 연결 및 동작(인증 토큰 발급, 시세 조회, 잔고 조회)은 모두 정상적으로 검증되었습니다.
> *   **[미해결 이슈]** 현재 등록된 모의투자 계좌의 가상 예수금이 `0원`으로 잡히는 현상이 있습니다. 이로 인해 매수/매도 주문이 실제로 전송되지 않고 안전하게 건너뛰기(Skip) 처리되고 있습니다. 한국투자증권 MTS나 홈페이지를 통해 해당 모의투자 계좌의 가상 자산 설정(참가 취소 후 재신청 등)을 완료해 주셔야 정상적인 매매 테스트가 가능합니다.

## 프로젝트 구조

*   `main.py`: 프로그램 시작점 (설정값 로드 및 실행)
*   `config.py`: 환경 변수 및 계좌 번호 파싱/검증
*   `auth.py`: OAuth 토큰 발급 및 로컬 파일 캐싱 (`token_cache.json`)
*   `api_client.py`: API 호출 공통 클라이언트 (인증 헤더 처리 및 재시도 기능)
*   `market_data.py`: 삼성전자 현재가 시세 조회
*   `account.py`: 계좌 예수금 및 삼성전자 보유 수량 조회
*   `orders.py`: 지정가 현금 매수/매도 주문 전송
*   `trader.py`: 매매 시간 검증 및 전체 트레이딩 루프 제어
*   `logger.py`: 로그 설정 (`trader.log` 파일 및 콘솔 동시 출력)

## 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 환경 변수 설정
프로젝트 루트 폴더에 `.env` 파일을 생성하고 아래 정보를 기입합니다. (보안을 위해 `.gitignore`에 등록되어 깃허브에는 업로드되지 않습니다.)

```env
GH_ACCOUNT="모의투자_계좌번호"
GH_APPKEY="모의투자용_AppKey"
GH_APPSECRET="모의투자용_AppSecret"
```

### 3. 프로그램 실행
```bash
python samsung_auto_trader/main.py
```

## 매매 핵심 로직

1.  평일 **오전 9시 10분 ~ 오후 3시 30분** 사이에만 작동합니다. (장외 시간 대기, 15:30 이후 프로그램 자동 종료)
2.  설정된 포링 주기(기본 30초)마다 삼성전자 현재가($P$)를 체크합니다.
3.  현재가 기준 지정가 주문 조건은 다음과 같습니다.
    *   **매수 주문**: 현재가 - 2,000원 (보유 예수금이 부족할 경우 경고 로그 출력 후 스킵)
    *   **매도 주문**: 현재가 + 2,000원 (보유 주식이 0주일 경우 스킵)
4.  주문 이후 계좌 상태를 즉시 재조회하여 실시간 체결 여부를 비교 및 로깅합니다.
